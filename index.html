Battles of Legend

Here's the link to, battlesoflegend.com This is a Full-Stack CRUD app using Angular.js, Express, Massive, and ElephantSQL. It's a turn based game where you select 2 combatants to battle to the death. I used a higher order function to join 2 tables from ElephantSQL. I also played around with a lot of functionality in both the controller and the service. I used ngAnimate to make it more visually appealing and I implemented some Math.methods to make the game more dynamic. I also threw in a canvas animation with jQuery to make the home view pop. You can create you're own unique hero! You can make you mother in-law a combatant if you felt so inclined, or perhaps you're favorite historical figure. Have fun with it!
